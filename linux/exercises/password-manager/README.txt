Password Manager
--------------------------------------------------------------
A simple and secure password manager written in Bash.




Setup
--------------------------------------------------------------
Open a terminal and run the following commands.

1. Run the setup script (requires root privileges):
 
  - sudo ./setup.sh
   

2. Log out and log back in to apply group changes.


 Usage
-------------------------------------------------------------
Run the password manager script:

 -  bash password_manager.sh       #Using the bash command
 - ./password_manager             # Run as executable

You will be prompted to create a master password. Please follow the instructions and proceed.
